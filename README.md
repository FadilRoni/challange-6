# Book Store API GoLang

Challange 6 GoLang

## How to use

```sh
  go run main.go
  ```

## Available Endpoints:

### Get All Books: ( GET Request )
* `localhost:8080/books`

### Get single Book ( GET request )
* `localhost:8080/book/:id`

### Add New Book: ( POST Request )
* `localhost:8080/book`

### Update Book: ( PUT Request )
* `localhost:8080/book/:id`

### Delete Book: ( Delete Request )
* `localhost:8080/book/:id`
